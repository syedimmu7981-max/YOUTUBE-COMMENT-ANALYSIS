# YouTube Comment Analysis

## Objective
Analyze YouTube comments and classify them as Positive, Negative, or Neutral.

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

## Project Workflow
1. Collect comments.
2. Load comments using Pandas.
3. Perform sentiment analysis using TextBlob.
4. Categorize comments.
5. Visualize sentiment distribution.

## Files
- main.py
- comments.csv
- requirements.txt

## How to Run
pip install -r requirements.txt

python main.py

## Output
- Sentiment classification table
- Bar chart showing sentiment distribution

## Conclusion
This project demonstrates basic sentiment analysis of YouTube comments.
